### Requirements

- Node ([install details](https://github.com/nvm-sh/nvm#nvmrc))
- Simulator (iOS or Android)


### Stack

The future stack:

- Typescript
- React Native
- Axios
- Styled Components
- ESlint
- Prettier
- Husky

---

### Initialization

Installing dependencies

```bash
yarn
```

Running on simulator

```bash
yarn ios
### or
yarn android
```
